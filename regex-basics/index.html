  • Database searches etc.

Pattern Matching

  • Matching a literal character
  • Using Meta-characters
    • Single Character
    • Quantifiers
    • Positional

Single character

  1. \d -> 0-9
  2. \w -> A-Za-z0-9
  3. \s -> Any whitespace
  4. . & *

Quantifiers

  1. * -> 0 or More
  2. + -> One or more (at least one)
  3. ? -> 0 or 1 (Optional)
  4. {min,max}

Positional

  1. ^ -> Start of line
  2. $ -> End of line
  3. \b -> Word boundary

Character Classes

  1. [abc] -> Match a or b or c
  2. [a-zA-z0-9] -> a to z Or A to Z Or 0 to 9
